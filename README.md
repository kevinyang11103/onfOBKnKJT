# Python图书管理系统 python62

## 项目概述

Python图书管理系统是一款使用Python语言开发，基于Django框架的图书管理应用。该系统致力于简化图书的借阅和归还流程，同时为管理员提供便捷的图书库存管理功能，以提高工作效率。

## 技术栈

- Python 3.6.1
- Django 2.2
- PyMySQL 0.9.3
- MySQL 5.7.23

## 功能模块

### 学生权限

- 图书查询
- 借阅图书
- 归还图书

### 管理员权限

- 图书库存管理
- 图书信息编辑
- 用户借阅信息查询

## 系统角色

- 学生：可进行图书查询、借阅与归还操作
- 管理员：负责图书库存管理、图书信息维护及用户借阅信息管理

## 运行环境

- 操作系统：兼容Windows、Linux和MacOS
- 数据库：MySQL 5.7.23及以上版本

## 部署步骤

1. 安装Python 3.6.1
2. 安装Django框架
3. 安装PyMySQL
4. 配置MySQL数据库
5. 导入系统初始数据
6. 运行Django开发服务器
